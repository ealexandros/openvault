# OpenVault Dev Container

Managed environment for **Rust**, **Node.js**, and **Tauri** development using **Bun**.

## Getting Started

1. **Reopen in Container**: Run the command via VS Code (bottom left corner or Command Palette).
2. **Wait**: Initial setup installs all dependencies automatically. (note: this might take a while)

## Workflow Tips

- **CLI Helpers**: Run `make help` to list all available project commands.
